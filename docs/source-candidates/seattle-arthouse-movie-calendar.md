Community-maintained ICS aggregator of Seattle arthouse cinema screenings at
`https://seattle-movies.innocence.com/film_calendar.ics`. Returns ~322 events.

Covers: The Beacon, Central Cinema, Grand Illusion, Meaningful Movies Ballard,
NW Film Forum, SIFF.

**Not viable:** All covered venues (except Meaningful Movies Ballard) are already
individual sources in this repo. Adding the aggregator would create duplicate events
in tag-based aggregate calendars (e.g., the Film tag would double-count screenings).
