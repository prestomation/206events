Sound Transit's public events calendar at soundtransit.org. Includes
community events, sports events (Mariners, Sounders, FIFA World Cup),
and public board/committee meetings. Uses a custom JSON API at
`/calendar_events_grouped.json` with pagination (`page=0,1,2,…`).
51 events confirmed in first build. Custom IRipper in
`sources/sound_transit/`.
