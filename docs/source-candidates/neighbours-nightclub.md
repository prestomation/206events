Seattle's oldest LGBTQ nightclub (est. 1983), 1509 Broadway, Capitol
Hill. Weekly recurring nights (Latin Night, drag, DJ sets).

Investigated 2026-07-16:
- Site (`neighboursnightclub.com`) is built on Wix and is fully
  client-side rendered — the static HTML response contains only Wix
  bootstrap JS, no event data, no schema.org markup, no discoverable
  REST/API endpoint.
- No ICS/iCal, no Eventbrite/DICE/AXS integration found.

**Verdict**: Not viable without browser automation, which the repo
doesn't currently do for this kind of source. Re-evaluate if the venue
migrates off Wix.
