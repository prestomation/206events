Legendary West Seattle record store and café at 4559 California Ave SW (Alaska Junction).

was derived from the DICE URL slug and tested in CI, but the DICE API returned 0 events and
1 parse error — the venueName filter is likely wrong. Other DICE venues in the repo use
names like "Kremwerk Complex", "The Vera Project", "The Black Lodge", "The Sunset Tavern"
(often with "The" or additional words not in the slug). DICE API and dice.fm are both
blocked from the remote execution environment; the correct venueName cannot be determined
without browser access to the DICE venue page.

Status: `candidate` — needs venueName verification before implementation. See
`docs/source-candidates/easy-street-records.md` for next steps.
