# Discovery Log — 2026-06-23

## Summary

Investigated 10+ potential sources across multiple Seattle verticals. Most were already in the repo, blocked, or not viable. Added Meany Center for the Performing Arts and Seattle Arts & Lectures as new custom HTML rippers.

## Added

### Meany Center for the Performing Arts
- **URL:** https://meanycenter.org/tickets/season
- **Platform:** Drupal 7 (no ICS feed)
- **Events:** 36 dated events from 25 productions (2026-2027 season, Oct–May)
- **Implementation:** Custom `IRipper` parsing the Drupal calendar view (`/tickets/events/calendar?date=YYYY-MM`) for ISO datetimes, cross-referenced with season listing for images/descriptions
- **Tags:** Music, Dance, Arts, University District
- **Candidate:** `docs/source-candidates/meany-center.md`

### Seattle Arts & Lectures
- **URL:** https://lectures.org/events/
- **Platform:** Custom HTML (CSS grid with `data-in-month` / `.short-date` structure)
- **Events:** 35+ upcoming events (2026–2027 season, Jul 2026–May 2027)
- **Implementation:** Custom `IRipper`; emits `UncertaintyError` for `startTime` on every event (listing page shows date only, not time)
- **Tags:** Books, Arts, Education
- **Candidate:** `docs/source-candidates/seattle-arts-lectures.md`
