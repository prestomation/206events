## Pioneer Square Market Network

Investigated `pioneersquaremarket.net` as a candidate source. The site is a
React SPA backed by Supabase (PostgREST). Extracted public anon JWT key from
the JS bundle and confirmed the `public_events` table is publicly accessible.

API returns 51 upcoming events: concerts (Baba Yaga, Hotel Crocodile area),
community events (Pioneer Park, Downtown Seattle Association), festivals
(Folklife 2026), jazz (Seattle Jazz Fellowship), and markets. Also includes
7 Vancouver BC FIFA World Cup matches which are filtered out by a state check
(`venue_location.state !== 'WA'`).

Implemented as `sources/pioneer_square_market/` using the `IRipper` interface
with direct Supabase PostgREST calls. Tags: Community, Pioneer Square, Music.
